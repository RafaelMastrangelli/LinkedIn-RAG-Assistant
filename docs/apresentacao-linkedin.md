Estou compartilhando um projeto que pode facilitar a vida de muitos desenvolvedores: o LinkedIn Job Bot - Telegram Notifier.
O problema que motivou esse projeto foi a necessidade de filtrar vagas de emprego no LinkedIn de forma eficiente, sem precisar lidar com anúncios pagos ou vagas irrelevantes.
Foram construídos recursos como:
- Scraping orgânico no LinkedIn
- Filtro de vagas promovidas
- Busca paralela por múltiplos cargos
- Bloqueio de stacks não desejadas
- Banco de dados anti-spam
Usando TypeScript, Node.js, Playwright, Cheerio, node-telegram-bot-api, better-sqlite3 e node-cron.
Aprendi bastante sobre automação web e integração com serviços de mensagens.
Quer ver como funciona? Acesse: https://github.com/seu-usuario/linkedin-job-bot
